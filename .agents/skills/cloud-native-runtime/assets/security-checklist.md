# Security Checklist

- [ ] Authn before protected access.
- [ ] Authz scoped to resource/tenant/domain.
- [ ] Boundary input validation.
- [ ] Least-privilege IAM.
- [ ] Secrets stored through approved secret mechanism.
- [ ] Encryption requirements satisfied.
- [ ] Audit/security logs do not expose sensitive values.
- [ ] Uploads, webhooks, queues, and schedules treated as untrusted.
