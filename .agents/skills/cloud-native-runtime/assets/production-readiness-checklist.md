# Production Readiness Checklist

- [ ] Requirements mapped to implementation evidence.
- [ ] Runtime and infra source of truth verified.
- [ ] Secrets/config available in target environment.
- [ ] Migrations or storage changes are reversible or recoverable.
- [ ] Logs, metrics, traces, and alerts cover rollout.
- [ ] Retry, backpressure, and idempotency are bounded.
- [ ] Security review complete for exposed or privileged surfaces.
- [ ] Validation commands passed.
- [ ] Rollback plan documented and executable.
