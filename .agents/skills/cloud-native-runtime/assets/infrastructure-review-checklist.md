# Infrastructure Review Checklist

- [ ] Correct account, region, stack, and environment.
- [ ] No unintended replacements, deletes, or public exposure.
- [ ] IAM is least privilege.
- [ ] Secrets use approved encrypted paths.
- [ ] Network access is scoped.
- [ ] Data resources have backup/lifecycle/deletion posture.
- [ ] Observability and alarms match runtime change.
- [ ] Preview or plan reviewed before apply.
