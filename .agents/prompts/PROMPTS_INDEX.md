## Usage Pattern

1. Start from a category README.
2. Copy a specific prompt file.
3. Fill in concrete task context.
4. Pair with `agent-execution-mode` and `repo-standards-enforcement` unless intentionally exploratory.

## Prompt Writing Rules

- Prefer imperative language.
- Require validation evidence.
- Require done criteria.
- Forbid placeholders and fake completion.
- Require explicit blocker reporting when blocked.
