# Repo Standards Check

Use this to enforce repository conventions before merge.

## Prompt

Run a repository standards compliance pass.

Requirements:
1. Detect active toolchain and conventions.
2. Verify package manager purity.
3. Verify lint/typecheck/test expectations.
4. Identify non-compliant files and required fixes.
5. Provide explicit command evidence.

Output:
- Standards detected
- Commands run and outcomes
- Violations found
- Remediation actions
