# Ecosystem Analysis

Use this to compare libraries/tools/framework options for adoption decisions.

## Prompt

Analyze the ecosystem options and recommend an adoption path.

Requirements:
1. Define evaluation criteria (fit, maturity, performance, DX, maintenance).
2. Compare top options with concrete pros/cons.
3. Include migration and lock-in considerations.
4. Provide a clear recommendation and fallback plan.
5. Call out unknowns requiring further validation.

Output:
- Criteria
- Option comparison matrix
- Recommendation
- Migration/fallback notes
