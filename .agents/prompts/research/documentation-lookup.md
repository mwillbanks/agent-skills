# Documentation Lookup

Use this for focused lookup of authoritative docs needed for a task.

## Prompt

Find and summarize the exact documentation needed to unblock implementation.

Requirements:
1. Prefer official documentation first.
2. Extract only relevant sections and constraints.
3. Highlight version-specific caveats.
4. Translate docs into concrete implementation guidance.
5. Flag any ambiguous or conflicting guidance.

Output:
- Sources used
- Key constraints
- Implementation guidance
- Open questions
