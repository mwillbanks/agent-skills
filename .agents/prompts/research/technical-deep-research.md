# Technical Deep Research

Use this for deep analysis that informs architecture or implementation choices.

## Prompt

Perform technical deep research and deliver implementation-ready conclusions.

Requirements:
1. Define research questions and decision criteria.
2. Gather evidence from high-quality sources.
3. Compare options against constraints and tradeoffs.
4. Produce recommendation with rationale.
5. Include implementation implications and risks.

Output:
- Question framing
- Evidence summary
- Option comparison
- Recommended path
- Risks and mitigations
