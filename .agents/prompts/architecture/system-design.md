# System Design

Use this for high-level architecture of systems and cross-cutting concerns.

## Prompt

Design a production-oriented system architecture.

Requirements:
1. Define goals, non-goals, and constraints.
2. Propose components, responsibilities, and interactions.
3. Identify failure modes and recovery strategies.
4. Specify observability and operational requirements.
5. Document tradeoffs and why this design is chosen.

Output:
- Architecture overview
- Component boundaries
- Data/control flow
- Tradeoff analysis
- Implementation implications
