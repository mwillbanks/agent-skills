# Implementation Prompts

Use these prompts when code must be fully implemented, not sketched.

## Available Prompts

- [production-implementation.md](production-implementation.md)
- [implementation-hardening.md](implementation-hardening.md)
- [architecture-implementation.md](architecture-implementation.md)

## Rule

Choose `production-implementation` by default, escalate to `implementation-hardening` when correctness or prior failure risk is high.
